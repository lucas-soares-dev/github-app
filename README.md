# Github App Vue.js

SPA desenvolvida com Vue.js 3, que permite a busca de repositórios do Github e exibe as informações mais relevantes, como: título, descrição, tecnologias, estrelas, issues, etc.

## Execução do projeto

```
$ yarn install
$ yarn dev

ou

$ npm install
$ npm run dev
```

## Execução dos testes

```
$ yarn test

ou

$ npm run test
```

## Tecnologias utilizadas

- Vue.js 3
- Bootstrap 4
- Pinia
- Axios
- FontAwesome
